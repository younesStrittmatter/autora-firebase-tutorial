Running a full closed loop

Introduction

We'll be running a full closed loop with autora and a super-experiment. The dependent variable will be the coherence (difficulty) of a task, the independent variable will be the accuracy. The documentation for the template we'll be using to set up the experiment can be found here: cra template. The documentation for the experiment we'll be using, can be found here: super experiment. (You don't need to checkout the documentation before the tutorial, all will be explained then)

Prerequisites

To save some time, we can prepare some things before the tutorial starts (if you run into any issues here or have questions, feel free to contact me)

Install node and npm

Install node and npm. A tutorial can be found here: Installing Node.js and npm

Prepare your autora repository

In the aurora repository:

  • Download this file: Download
  • Checkout the following branch: 236-feat-set-up-initial-experiment-server-and-cycle-on-firebase
  • Copy the downloaded file experiment_runner_demo.py into the root of your repository
  • You'll also need to install an additional dependency: pip install firebase-admin
  • If you want to make change to the theorist/experimentalist, you can modify experiment_runner_demo.py

During the tutorial

This is a little intro of thing, we'll be doing during the tutorial to get started (no need to do anything here, all will be explained during the tutorial)

  • Create a new folder/project (for example in pycharm) or a different ide for the experiment (you can also do this from the terminal but an ide is recommended). This will be a different project then your autora copy.
  • To get the template run: npx create-react-app my-autora-experiment --template autora-firebase
  • In the folder there will now be a README.md file. Follow the steps listed there.
  • The experiment itself will be a version of the super-experiment. Follow the steps described in the documentation using the condition from autora to create the settings for the experiment and the return value as observation
  • Deploy the experiment to firebase (as described in the README.md of the template)
  • In the autora repository: add the firebase-credentials from your firebase project (-> project -> project settings -> service accounts -> generate new private key)
  • Run the experiment_runner_demo.py
  • Get the link in for the project from firebase project and share it in the zoom chat
  • Participate in the experiments of your team members to provide them with data
  • After running the experiment (enough data is collected) your autora runner should give you a new model for your data
